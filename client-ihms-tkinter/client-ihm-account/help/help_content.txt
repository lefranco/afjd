== Création de nouveau compte ==

Entrer le psuedo, qui doit être différent de ceux qui existent déjà.
(Pas d'accents pour le pseudo)
Répeter deux fois le mot de passe initial à l'identique
L'adresse email doit être valide, un mail de confirmation sera envoyé, avec un code de quatre chiffre.

Le numéro de téléphone, le nom de famille et le prénom sont facultatifs.
Le pays et le fuseau horaire seront visibles sur le site.

Cliquer "Créer le compte"

== Modification de compte existant ==

Entrer le pseudo, qui doit être différent de ceux qui existent déjà et cliquer "M'identifier"
Les autres actions deviennent possibles.

Entrer le code de quatre chiffres reçu par email et cliquer "Valider mon adresse email"
Modifier certaines informations et cliquer "Mettre à jour"
Entrer un nouveau ot de passe (répété deux fois à l'idetntique) et cliquer "Changer de mot de passe"

La suppression du compte n'est pas possible tant que le joueur est dans une partie.
