== Recharger la liste des parties ==
Met à jour la liste des parties avec les informations du serveur. 

== Recharger la partie ==
Met à jour la partie avec les informations du serveur. 
Se place sur la saison en cours
Met à jour la carte, les messages et les déclarations
Entre en mode partie

*********
Initialement l'interface est en mode partie.
*********


== Editer les possessions de centres ==
Change de mode pour pouvoir modifier les possessions de centres sur la carte.
Invalide (rend impossible) la soumission des ordres
Quitte mode partie

== Editer les positions des unités ==
Change de mode pour pouvoir modifier la position des unités sur la carte.
Invalide (rend impossible) la soumission des ordres
Quitte mode partie

== Retour à l'entrée d'ordres ==
Change de mode pour pouvoir entrer les ordres sur la carte
Ne revient pas en mode partie, (la soumission des ordres reste impossible), il faut recharger la partie du serveur.

== Simuler ==
Interroge directement le moteur de résolution avec la position et les ordres sur la carte.
Ne nécessite aucun droit.
Ne nécessite pas le mode partie.

== Réinitialiser ==
Effece tous les ordres entrés jusqu'à présent

== Soumettre ==
Soumet les ordres tels qu'entrés sur la carte pour la partie en question au moteur de partie. Si ces ordres sont acceptés par le moteur de résolution, ils seront enregistrés.
Nécessite les droits du rôle
Nécessite le mode partie.
(il est possible de soumettre les ordres des autres joueurs, ils seront refusés par le serveur)

== Envoyer ==
Enregistre un message à destination de l'interlocuteur. L'envoie sur le serveur de messages.
Nécessite les droits du rôle
Nécessite une partie avec négociation.
Nécessite le mode partie.

== Publier ==
Enregistre une déclaration lisible par tous. L'envoie sur le serveur de déclarations.
Nécessite les droits du rôle
Nécessite une partie avec négociation.
Nécessite le mode partie.

== Rectifier ==
Soumet la position entrée sur la carte pour la partie
Nécessite les droits admin.
Nécessite le mode partie.

== Effacer ==
Vide la fenêtre de log des événements.
